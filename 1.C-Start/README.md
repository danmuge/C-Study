### 목표: hello world찍기

- 1.컴퓨터를 킨다
- 2.비주얼 스티디오 들어간다
- 3.파일을 만든다.
- 4.맨위에 include<stdio.h>
- 5.int main 쓴다
- 6.printf("hello world");를 쓴다.
- 7.ctrl+Alt+c 하고 gcc컴파일러를 이용해서 빌드한다.
- 8.ctrl+Alt+n을 누른다.
- 9.실행된다
  빌드 = 링크+컴파일러(실행파일을 만들어줌)
  컴파일러=컴파일을 시켜주는 도구
